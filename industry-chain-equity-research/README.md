# Industry Chain Equity Research Skill

## 内容简介

这是一个面向科技与 AI 产业链投研的 Codex skill。它的目标不是直接给股票池，也不是根据题材联想熟悉公司，而是把一个产业催化拆成可验证的产业链、供应链、利润池和估值框架。

适合的问题包括：

- AI 服务器、GPU、HBM、CoWoS、光模块、PCB、MLCC、液冷、电力设备等硬件链条怎么拆。
- 某个上游瓶颈，如 HVLP 铜箔、电子布、覆铜板、先进封装材料，应该如何找到 A 股映射标的。
- 某条新闻或产业变化，应该选系统厂、零部件、材料、设备，还是应用层。
- AI 产业链中哪些层级最赚钱，哪些层级最容易被卷。

## 核心方法

skill 采用“美国/全球为主，中国为辅”的框架：

1. 先看全球主导玩家和真实利润池。
2. 再看中国公司是国产替代、产能跟随、供应链配套，还是应用本地化。
3. 不强行找中国 proxy；如果利润池主要在海外，也要如实标注。
4. 所有候选公司必须有产品、客户、产能、认证、订单、价格或财报证据支撑。

## AI 八层框架

skill 内置的研究分层包括：

| 层级 | 核心问题 |
|---|---|
| 半导体设备 / EDA / 材料 | 谁提供先进芯片制造工具 |
| 晶圆代工 / 先进封装 | 谁解决制程和 CoWoS 等封装瓶颈 |
| AI GPU / 加速器 | 谁掌握训练与推理算力 |
| HBM / 内存 / 存储 | 谁解决内存带宽和数据供给 |
| 服务器 / 网络 / 液冷 / 电力 | 谁把芯片部署成可运行集群 |
| 云平台 / IaaS | 谁把算力卖给模型公司和企业 |
| 基础模型 / API | 谁把算力转化为通用智能能力 |
| 应用 / Agent / SaaS | 谁把 AI 嵌入具体工作流并收费 |

## 输出物

使用这个 skill 后，理想输出不是一句“看好某某公司”，而是一份结构化研究表：

- 产业链挖掘地图
- 产业链节点搜索记录
- 候选标的对比表
- 利润池与被卷风险排序
- 关键变量仪表盘
- 供需测算
- 价格、成本与利润弹性
- 三年情景预测
- 估值比较
- 关键假设与反证清单

## 使用方式

在 Codex 中显式调用：

```text
Use $industry-chain-equity-research to research the A-share opportunities from 800V HVDC AI data center power architecture.
```

也可以用于中文问题：

```text
调用投研 skill，研究 HVLP4 铜箔缺货对应的 A 股产业链标的。
```

## 适用场景

- 从一条产业新闻拆出投资方向。
- 从海外龙头上涨寻找中国供应链映射。
- 从一个上游材料涨价寻找下游和同类扩散。
- 从 AI capex、GPU、HBM、CoWoS、液冷、电力等变量判断利润池。
- 对候选公司做供需、利润和估值验证。

## 不适合做什么

- 不适合直接输出买入建议。
- 不适合把熟悉股票硬塞进题材。
- 不适合在没有数据的情况下判断“最受益”。
- 不适合替代实时行情、财报和公告核验。
